# 🛒 Dengue Prevention Marketplace - Feature Guide

## 📋 Overview
The Dengue Prevention Marketplace is a comprehensive e-commerce platform integrated into the dengue risk prediction system, allowing users to purchase WHO-approved dengue prevention products.

## 🔧 Technical Implementation

### 🗄️ Database Models
- **ProductCategory**: Categories with icons and descriptions
- **Product**: Complete product information with pricing, ratings
- **Cart**: User shopping cart management
- **Order**: Order processing and tracking
- **OrderItem**: Individual items in orders

### 🎨 Frontend Features
- **Responsive Design**: Bootstrap 5 with mobile-first approach
- **AJAX Cart**: Real-time cart updates without page reload
- **Product Search**: Filter by category and search by name
- **Professional UI**: Clean, modern e-commerce interface

### 🔒 Security & Payment
- **Cash on Delivery**: Secure payment method for demo
- **User Authentication**: Integrated with existing login system
- **Order Tracking**: Complete order history and status

## 🚀 Access Points

### 📱 Navigation Integration
1. **Main Navbar**: "Prevention Store" in Tools dropdown
2. **Dashboard**: "Prevention Store" button in tools section
3. **Cart Icon**: Top-right with live item count badge
4. **Direct URL**: `/marketplace`
